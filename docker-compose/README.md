# Demo Docker Compose
## Commands
### Start
```sh
# as console
docker-compose up

# as daemon
docker-compose start
```

### Stop
```sh
docker-compose stop
```

## URLs
- http://localhost:8080/
- http://localhost:8080/node/ <= Trailing slash needed
- http://localhost:8080/node/api
- http://localhost:8080/node/api/anyword
